# License Policy

This repository uses a mixed-artifact license policy because RealMythos will publish documentation, generated dataset artifacts, scripts, reports, and possibly model-related materials.

## Planned Licenses

| Artifact Type | Planned License |
|---|---|
| Dataset annotations, reasoning traces, metadata, and curation artifacts | CC-BY-4.0 |
| Technical reports and documentation | CC-BY-4.0 |
| Project scripts and tooling | Apache-2.0 |
| Model artifacts | To be finalized before Stage 2 release |

## Upstream Source Code

Some dataset records may include source code snippets derived from upstream open-source projects. Those snippets may remain subject to their original upstream licenses. RealMythos does not claim to relicense upstream project code.

## Before Public Release

Before the first public release, this file should be updated with:

- Exact license identifiers
- Links to full license texts
- Hugging Face dataset license metadata
- Any additional restrictions required by upstream data sources
