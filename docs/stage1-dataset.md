The Hugging Face repository hosts the dataset files and dataset card. The GitHub repository hosts release documentation, responsible-use notes, and reproducibility-oriented pipeline code. The technical report draft will be hosted through Google Drive during pre-arXiv iteration.

## Research Lineage

This release is influenced by two earlier lines of work: **Reef: A Framework for Collecting Real-World Vulnerabilities and Fixes**, our ASE 2023 work on vulnerability/fix collection, and **API-guided Dataset Synthesis to Finetune Large Code Models**, our OOPSLA 2025 work on structured code-data synthesis. RealMythos extends these ideas toward reasoning-data construction, PoC-aware evaluation, and reproducible release infrastructure.

The RealMythos technical report will also be made available as an arXiv preprint; the arXiv link will be added once available.
