# Stage 2 Model

Stage 2 will release an open security reasoning model trained using the Stage 1 dataset.

Internal experiments indicate an average performance improvement of more than 25% over the selected baseline setup. Public claims will be finalized only after the evaluation configuration, baseline definitions, and reproducibility artifacts are ready.

## Planned Release Materials

- Model checkpoint or adapter
- Model card
- Training configuration
- Evaluation scripts
- Baseline comparison report
- Inference examples
- Safety and limitation notes

## Dependency on Stage 1

The Stage 2 release should link to the exact Stage 1 dataset version used for training.
